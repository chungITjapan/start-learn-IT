Đây là nội dung trang web

Đây là một ví dụ về cách sử dụng thẻ nav và a để tạo menu trong HTML.Trang web này có một menu đơn giản với năm liên kết đến các trang khác nhau trên trang web.Khi người dùng nhập vào liên kết này,trang web sẽ chuyển hướng đến trang đó.
